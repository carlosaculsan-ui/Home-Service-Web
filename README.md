Hanap.ph is a home services marketplace connecting Metro Manila customers with verified taskers across 6 categories (Cleaning, Carpentry, Electrical, Aircon, Painting, Plumbing). It handles the full service lifecycle — booking, PayMongo payments (GCash, PayMaya, cards), e-wallet earnings, dispute resolution, and reviews. Built with React 19, Node.js/Express, Supabase, and Tailwind CSS, with Jitsi video calls, AI support via Groq LLaMA 3.3, and email notifications.
